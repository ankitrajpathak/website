# Ankit Raj Pathak — Portfolio

Personal portfolio site. One page. No frameworks. Just HTML, CSS, and intent.

[![Launch Site](https://img.shields.io/badge/🚀_Visit_Site-ankitrajpathak.github.io-7EB8F7?style=for-the-badge&labelColor=060810)](https://ankitrajpathak.github.io/website/)

---

### What's inside

- **Hero** — animated intro with profile card and CTAs
- **About** — background and working philosophy
- **Experience** — timeline of roles at Zeron, Vehere, I-PAC
- **Portfolio** — 9 video projects with click-to-play (Google Drive embeds), filterable by category
- **Skills** — strategy, performance, and craft breakdown + tool stack
- **Education & Certs** — degrees and certifications
- **Contact** — email, phone, socials

### Tech

Single `index.html` file. Zero dependencies. Zero build step.

- **Fonts** — Syne (display) + DM Sans (body) via Google Fonts
- **Design** — dark glassmorphism, ambient orbs, grain overlay, scroll-reveal animations
- **Video** — lazy-loaded Google Drive embeds (click thumbnail to play)
- **Responsive** — 3-col → 2-col → 1-col grid breakpoints
- **Custom cursor** — dot + ring with hover scaling

### Deploy

Drop `index.html` into any static host. It's one file.

```
# GitHub Pages
git add index.html
git commit -m "update portfolio"
git push
```

### Structure

```
website/
├── index.html    ← everything lives here
└── README.md
```
